# Photography Website

## About

This is a simple photography website that I created using HTML and CSS. The website displays different photography images, a short description about the website, and a list of my favorite photography styles.

I built this project to practice creating web pages, styling them with CSS, using Git for version control, uploading my work to GitHub, and deploying it with GitHub Pages.

## Features

- Home section
- About section
- Photo gallery
- Favorite photography styles
- Responsive navigation
- Website design sketch

## Technologies Used

- HTML
- CSS
- Git
- GitHub
- GitHub Pages
